### 3.1 回测框架

| 框架 | 特点 | 推荐度 |
|------|------|--------|
| **backtrader** | 功能全面，社区活跃 | ⭐⭐⭐⭐ |
| **vectorbt** | 向量化回测，速度快 | ⭐⭐⭐⭐ |
| **freqtrade** | 开箱即用，自带实盘 | ⭐⭐⭐ |

```python
# backtrader 简单示例
import backtrader as bt

class MyStrategy(bt.Strategy):
    params = (('sma_period', 20),)
    
    def __init__(self):
        self.sma = bt.indicators.SMA(self.data.close, period=self.p.sma_period)
    
    def next(self):
        if not self.position:
            if self.data.close[0] > self.sma[0]:
                self.buy()
        else:
            if self.data.close[0] < self.sma[0]:
                self.sell()

cerebro = bt.Cerebro()
cerebro.addstrategy(MyStrategy)
# 加载数据、运行回测...
cerebro.run()
```

### 3.3 风险管理

| 概念 | 说明 | 实现方式 |
|------|------|---------|
| **止损** | 单笔最大亏损 | 固定百分比 / ATR 动态止损 |
| **仓位管理** | 单笔占总资金比例 | 固定比例 / Kelly 公式 |
| **最大回撤限制** | 策略停止条件 | -20% 暂停策略 |
| **分散化** | 多币种/多策略 | 相关性分析 |

```python
# 仓位管理示例
def calculate_position_size(capital, risk_per_trade, entry_price, stop_loss):
    """
    风险百分比法：每笔交易最多亏损总资金的 1%
    """
    risk_amount = capital * risk_per_trade  # 1% 的资金
    price_risk = abs(entry_price - stop_loss)
    position_size = risk_amount / price_risk
    return position_size

# 示例: 10000 USDC, 1% 风险, ETH $3000 买入, $2850 止损
size = calculate_position_size(10000, 0.01, 3000, 2850)
# 结果: 0.667 ETH (最多亏损 $100)
```

### 3.4 回测注意事项

```
⚠️ 常见陷阱:

1. 过拟合: 策略在历史数据表现完美，实盘却亏损
   → 解决: 样本外测试、Walk-forward 分析

2. 幸存者偏差: 只回测存在的币种，忽略已退市的
   → 解决: 使用完整历史数据

3. 忽略滑点和手续费: 回测利润虚高
   → 解决: 加入 0.1% 手续费 + 0.05% 滑点

4. 未来函数: 使用了未来才知道的数据
   → 解决: 确保信号只使用当前及之前的数据
```
